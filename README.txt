PYTHON ОТ POWERSHELL - ЗАДЪЛЖИТЕЛНО!
--------------------------------------
Python 3.14 заседва при стартиране от PowerShell защото чака stdin.
ВИНАГИ стартирай Python скриптове така:

  cmd /c "C:\Python314\python.exe -u script.py < nul"

НИКОГА не използвай:
  C:\Python314\python.exe script.py        <- ЗАСЕДВА
  python script.py                         <- ЗАСЕДВА

За бърз достъп използвай:
  C:\XAUUSD_AI_Bridge\RUN_PY.bat script.py
